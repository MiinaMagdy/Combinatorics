Permutations

Definition

A permutation of nnn distinct objects is an ordering of these objects. The number of permutations of nnn distinct objects is n!n!n!. The number of permutations of nnn objects taken kkk at a time is n!(n−k)!\frac{n!}{(n-k)!}(n−k)!n!​.

Formula

P(n,r)=n(n−1)(n−2)⋯(n−r+1)=n!(n−r)!P(n,r)=n(n-1)(n-2)\cdots(n-r+1)=\frac{n!}{(n-r)!}P(n,r)=n(n−1)(n−2)⋯(n−r+1)=(n−r)!n!​.

Example

Elodie is putting on a fashion show and has five fabulous outfits for her five fabulous fashion models. However, on the day of the show, two of the outfits were ruined in an unfortunate permanent marker incident. Regardless, the show must go on and the remaining outfits will be presented. If each outfit can only be worn by one model and there is no time for any model to wear more than one dress, how many different shows can Elodie put on? (Note: Two shows are considered the same if they contain the same models wearing the same dresses.)

(A) 120(B) 60(C) 720(D) 5040(E) 24(F) 6\mathrm{(A)}\,120\quad\mathrm{(B)}\,60\quad\mathrm{(C)}\,720\quad\mathrm{(D)}\,5040\quad\mathrm{(E)}\,24\quad\mathrm{(F)}\,6(A)120(B)60(C)720(D)5040(E)24(F)6

Solution ✅

Answer: (B)\mathrm{(B)}(B)

Since two of the outfits are ruined, we only have three outfits. There are five models available for the first outfit, four models available for the second outfit, and three models available for the third outfit. Therefore, there are 5⋅4⋅3=605 \cdot 4 \cdot 3 = \boxed{60}5⋅4⋅3=60​ ways in which the models can be matched to the outfits.

Combinations

Definition

A combination, sometimes called a binomial coefficient, is a way of choosing rrr objects from a set of nnn where the order in which the objects are chosen is irrelevant. We are generally concerned with finding the number of combinations of size rrr from an original set of size nnn

Formulas/Identities

  • (nr)=n!r!(n−r)!{{n}\choose {r}} = \frac {n!} {r!(n-r)!}(rn​)=r!(n−r)!n!​

  • (n−1r−1)+(n−1r)=(nr)\binom{n-1}{r-1}+\binom{n-1}{r}=\binom{n}{r}(r−1n−1​)+(rn−1​)=(rn​)

Techniques

Casework

Definition

casework is a counting method that involves splitting a problem into several parts, counting these parts individually, then adding together the totals of each part. Casework is a very general problem-solving approach, and as such has wide applicability.

Example

How many positive integers satisfy the equation x4+y<70x^4 + y < 70x4+y<70?

Solution ✅

We use casework, based on the value of x.

Case 1: x=1x=1x=1. Then this problem becomes y<69y<69y<69, so there are 686868 possible values for yyy, and thus 686868 solutions when xxx is one.

Case 2: x=2x=2x=2. Then this problem becomes y<54y<54y<54, so there are 535353 possible values for yyy, and thus 535353 solutions when xxx is two.

If x=3x=3x=3, the problem becomes y<−11y<-11y<−11. But the question mandates that yyy is positive, so there are no solutions when y≥3y \geq 3y≥3. Thus, there are 68+53=12168+53=12168+53=121 positive integer solutions to the equation. □\square□

Complementary counting

Definition

complementary counting is a counting method where one counts what they don't want, then subtracts that from the total number of possibilities

Example

How many positive integers less than 100100100 are not a multiple of five?

Solution ✅

We use a complementary approach. The total number of positive integers, with no restrictions, is 999999 integers. What we don't want are the multiples of five. These are 5,10,...,955, 10,..., 955,10,...,95 or 1⋅5,2⋅5,...,19⋅51 \cdot 5, 2 \cdot 5,..., 19 \cdot 51⋅5,2⋅5,...,19⋅5; it's easy to see that there are 191919 of them. Thus, our answer is is 99−19=8099-19 = 8099−19=80. □\square□

Pigeonhole Principle

Definition

the pigeonhole principle states that if n+1n+1n+1 or more pigeons are placed into nnn holes, one hole must contain two or more pigeons

Example

  1. If a Martian has an infinite number of red, blue, yellow, and black socks in a drawer, how many socks must the Martian pull out of the drawer to guarantee he has a pair?

    Solution ✅

    The Martian must pull 5 socks out of the drawer to guarantee he has a pair. In this case the pigeons are the socks he pulls out and the holes are the colors. Thus, if he pulls out 5 socks, the Pigeonhole Principle states that some two of them have the same color. Also, note that it is possible to pull out 4 socks without obtaining a pair.

  1. Suppose SSS is a set of n+1n + 1n+1 integers. Prove that there exist distinct a,b∈Sa, b \in Sa,b∈S such that a−ba - ba−b is a multiple of nnn.

    Solution ✅

    Consider the residues of the elements of SSS, modulo nnn. By the Pigeonhole Principle, there exist distinct a,b∈Sa, b \in Sa,b∈S such that a≡b(modn)a \equiv b \pmod na≡b(modn), as desired.

Find m+n.m+n.m+n.

Solution1 ✅

Consider the three-digit arrangement, aba‾\overline{aba}aba.
